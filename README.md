# YLD Tech challenge

## Steps to run & test

### 1. Clone repo
`git clone git@github.com:AimeurAmin/yld-tech-challenge.git`

### 2. access package folder
`cd package`

### 3. install dependencies
`npm i`

### 4. build the library
`npm run build`

### 5. access test project folder
`cd ../test`

### 6. install library locally (link)
`npm link ../package`

### 7. run test project
`npm run dev`
